# Летний практикум по автоматизации процессов и администрированию (2026)

Данный репозиторий является основным источником учебных материалов, исходного кода, технической документации и практических заданий в рамках летнего курса.

## Технологический стек курса

* **Python (v3.10+)**: Разработка скриптов автоматизации, построение парсеров, интеграция с внешними API.
* **Linux & Docker**: Развертывание, конфигурирование и администрирование изолированных серверных сред.
* **n8n (Self-Hosted)**: Оркестрация сервисов, проектирование событийно-ориентированных архитектур и автоматизация бизнес-процессов.
